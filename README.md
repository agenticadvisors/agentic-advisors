# Agentic Advisors — Website

## Files
- `index.html` — the full site
- `logo-full.svg` — horizontal logo (nav + footer)
- `icon.svg` — icon mark (browser tab)
- `ben-fullbody.png` — founder photo

## To update the site
Edit `index.html` and re-upload to GitHub. Changes go live in ~60 seconds.

## To add Loom videos
Search for `onclick="alert(` in index.html — replace those placeholder divs with Loom iframes.
